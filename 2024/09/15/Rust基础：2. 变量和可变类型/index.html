Rust基础：2. 变量和可变类型

Posted on 2024-09-15

1. 变量

rust 的变量包括可变变量和不可变变量，默认是不可变变量

1.1 不可变变量

1.2 可变变量

2. 常量

  • 需要显式指定数据类型
  • 一般使用大写字母，否则会报 Warning
  • 必须使用常量表达式进行赋值，即必须是编译期能计算出的值
  • 常量不支持重定义（遮蔽）

常量和不可变变量区别

  • 常量通过 const 声明，变量通过 let 声明
  • 常量在编译期就确定了值，不可变变量在运行期确定
  • 常量不可修改，不可变变量可以通过 shadowing 修改
  • 常量不需声明类型，不可变变量不需要
  • 常量一般使用大写字母，不可变变量不需要

3. 静态变量

  • static 关键字声明，编译时分配内存，并且在程序运行期间一直存在
  • 整个程序共享，多个线程之间共享
  • 程序运行期间不会改变，通常用于存储全局状态

与常量的相似之处

  • 一个使用 static，一个使用 const
  • 名称都要求大写，否则 warning
  • 必须指定类型
  • 常量表达式进行赋值
  • 都不支持重定义（遮蔽）

与常量的区别

  • 常量编译时被内联，静态变量只有一个内存地址
  • 静态变量可通过 mut 关键字定义为可变

4. 作用域和遮蔽

变量值在被{}包围的代码块中生存，被称作作用于。
另外也允许通过重新声明的方式进行变量遮蔽(variable shadowing)。
