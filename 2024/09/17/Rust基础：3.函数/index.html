Rust基础：3.函数

Posted on 2024-09-17

1. 函数组成

  • 声明函数关键字 fn
  • 参数名 add()
  • 参数 i 和 j 的参数类型 i32
  • 返回值类型 i32
  • 函数体 i + j

rust 不关心调用前还是调用后声明的函数

2. 函数名

  • 开头是字母或下划线，后面是数字、下划线、字母
  • 不能仅有下划线
  • 函数名和变量名使用蛇形命名法

3. 函数返回

  • return+分号，或没有 return+没有分号
  • 无返回值：不写返回类型，或返回()
  • 函数永不返回，用!作为返回类型
