Rust基础：4.流程控制

Posted on 2024-09-17

1. 条件控制

必须显式地使用 bool 值作为 if 的条件

在let语句中使用if

  • if 的每个分支可能得返回值都必须是相同类型
  • 代码块的值是其最后一个表达式的值

2. 循环

rust 有三种循环：loop、while和for

2.1 loop

loop 循环会一直执行下去，除非被 break 语句打断

循环可以返回值，使用break语句

多层循环之间消除歧义
break将只跳出内层循环，break 'counting_up';语句将退出外层循环

2.2 while

当条件为true，执行循环；当条件不再为true，退出循环

2.3 for

使用 for 遍历集合

所有权转移

通过索引和值的方式迭代数组
